# Manga-Downloader

A Lightweight Downloader for 漫画柜 and more.

## 支持的漫画源

- [漫画柜](https://www.manhuagui.com/)
- To Be Continued

## 从源码构建

### 第一步: 安装依赖

安装 [Go](https://go.dev/dl/) 1.25.0 或更高版本。

### 第二步: 克隆代码

```bash
git clone https://github.com/we1sper/Manga-Downloader.git
```

### 第三步: 构建

| 漫画源 | 构建命令                                                   |
|-----|--------------------------------------------------------|
| 漫画柜 | `cd Manga-Downloader && go build -o mhg.exe ./cli/mhg` |
